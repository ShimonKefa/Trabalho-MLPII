##**Descrição do Projeto**

Este projeto consiste em um sistema de console (CLI) em Python para gerenciamento de credenciais (logins e senhas).

O sistema permite:
- Adicionar credenciais
- Listar credenciais
- Buscar por site/app
- Deletar credenciais

As senhas podem ser:
- Inseridas manualmente
- Geradas automaticamente com critérios de tamanho

Além disso, o sistema aplica conceitos como:
- Programação Orientada a Objetos (POO)
- Herança e herança múltipla
- Uso de `@dataclass`
- Tratamento de exceções personalizadas
- List Comprehensions
- Estrutura `match/case`
